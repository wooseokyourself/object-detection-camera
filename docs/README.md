## Dependencies
- Raspbian with Python 3.3 or later
- OpenCV 4.4 or later
- CMake 2.8 or later

## Build
~~~
$ chmod 755 build.sh
$ ./build.sh
~~~

## Run
~~~
$ python3 app.py
~~~
